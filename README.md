# this-is-a-new-idea-r2

Turn rough concepts into structured briefs, clarifying questions, and quick viability scores.

## Quick Start

```bash
chmod +x ./scripts/run.sh
./scripts/run.sh brief "An app that helps parents coordinate carpools"
./scripts/run.sh score --format json "An app that helps parents coordinate carpools"
```
